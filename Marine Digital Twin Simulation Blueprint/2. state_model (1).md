## Spatial Representation — Zone Grid

The hull is divided into **50–200 rectangular patches called zones**. Each zone is an independent unit with its own state, updated separately every timestep.

**Why zones and not one averaged object?** The bow crashes into waves at high speed — maximum coating erosion. The keel sits in slow-moving deep water — maximum biofouling. The waterline gets UV from the sun. Averaging all of this together gives meaningless numbers that describe nowhere in particular on the ship.

```
BOW ─────────────────────────────────────────── STERN

WATERLINE ROW
  Z01   Z02   Z03   Z04   Z05   Z06   Z07   Z08

MID-DEPTH ROW
  Z09   Z10   Z11   Z12   Z13   Z14   Z15   Z16

KEEL ROW
  Z17   Z18   Z19   Z20   Z21   Z22   Z23   Z24
```

Each zone stores its own: `corrosion_depth` · `coating_thickness` · `barnacle_density` · `flow_velocity` · `roughness_index` · `risk_score`

## Zone Data Structure

Every zone is an instance of this object. Every field either comes from a real measurement, is computed from other fields, or feeds into another calculation. Nothing is decorative.

```python
Zone {
    # Identity fields
    zone_id          : string    -- e.g. 'Z_045'
    position         : (x, y, z) -- metres from ship reference point
    area             : float     -- m², used in drag and fuel calculations
    zone_type        : string    -- bow / midship / stern / keel / waterline

    # Corrosion (how deep has the metal been eaten)
    corrosion_depth  : float     -- mm of total metal lost so far
    corrosion_rate   : float     -- mm/year, recomputed each timestep

    # Coating (is the paint still protecting the hull)
    coating_thickness : float    -- micrometres (µm) remaining
    coating_intact    : boolean  -- True if thickness > failure threshold (default 15 µm)
    coating_type      : string   -- e.g. 'epoxy_antifouling_v2'

    # Biofouling (what has attached to the hull surface)
    barnacle_density  : float    -- count per m²
    slime_thickness   : float    -- micrometres of biofilm
    fouling_stage     : string   -- clean / slime / barnacle / heavy

    # Environment (refreshed every timestep from the environment database)
    local_flow_velocity : float  -- m/s
    local_temperature   : float  -- degrees Celsius
    local_oxygen        : float  -- mg/L dissolved oxygen
    local_salinity      : float  -- PSU practical salinity units

    # Computed outputs (calculated at end of each timestep)
    roughness_index   : float    -- Ra µm, from corrosion + fouling
    drag_coefficient  : float    -- dimensionless, drives fuel penalty
    risk_score        : float    -- 0.0 (healthy) to 1.0 (critical)
    last_validated    : date     -- date of last real inspection correction
}
```

## Four Update Rules — how zone state changes over time

These rules form a chain. Rule 1 is the most important because it controls everything else.

```
RULE 1  Coating = Master Switch
        If coating is intact  →  corrosion slows 95%, fouling slows 90%
        If coating fails      →  both run at full speed
            ↓
RULE 2  Corrosion → Roughness
        Every mm of metal lost makes surface rougher
        roughness_index = corrosion_depth × roughness_factor
            ↓
RULE 3  Fouling → Drag
        More barnacles = more drag = more fuel burned
        drag = base_drag × (1 + barnacle_density × drag_per_barnacle)
            ↓
RULE 4  Flow → Coating Wear
        Faster water flow erodes paint faster
        coating_loss = erosion_k × velocity² × time_step
            ↺ (Feedback loop back into Rule 1)
```

### Rule 1 — Coating is the master switch

```python
IF zone.coating_intact == True:
    corrosion_rate  = base_corrosion_rate * 0.05   # coating blocks 95% of corrosion
    biofouling_rate = base_fouling_rate   * 0.10   # antifouling paint blocks 90% of growth
ELSE:
    corrosion_rate  = k * oxygen * salinity_factor * temperature_factor
    biofouling_rate = base_rate * temperature_factor * harbour_idle_factor

# k = material rate constant from the quantum parameter table (material_table.json)
```

### Rule 2 — Corrosion makes the surface rougher

```python
zone.roughness_index += zone.corrosion_depth * roughness_factor
# roughness_factor is material-specific, from the quantum material table
```

### Rule 3 — Biofouling increases drag (and fuel cost)

```python
zone.drag_coefficient = base_drag * (1 + zone.barnacle_density * drag_per_barnacle)

# Fouling stage thresholds:
# 0 /m²       → clean
# 0–10 /m²    → slime
# 10–100 /m²  → barnacle
# 100+ /m²    → heavy
```

### Rule 4 — Fast water flow erodes the coating

```python
mechanical_loss = erosion_constant * flow_velocity**2 * time_step
chemical_loss   = coating_chemical_decay_rate * time_step
uv_loss         = uv_decay_rate * uv_index * time_step  # above waterline only

zone.coating_thickness -= (mechanical_loss + chemical_loss + uv_loss)
zone.coating_intact     = (zone.coating_thickness > 15)  # 15 µm = failure threshold
```
